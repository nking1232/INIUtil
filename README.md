# INIUtil
A very simply INI utility written in c++

<h2>#Features</h2></br>
-Cross Platform
-Writing to ini</br>
-Reading from ini</br>
-Sorting keys and values into groups.</br>
-Comments</br></br>
-Nesting groups inside of groups</br>

<h3>#Unsupported</h3>
-One liners

<h3>#Notes</h3>
When compiling a debug build define __DEBUG__.
